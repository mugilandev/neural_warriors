# Agri-Solve Pro - Project Analysis & Navigation Guide

## Project Overview
Agri-Solve Pro is a full-stack Progressive Web App (PWA) for agricultural crop diagnostics.
Built with React + TypeScript + Tailwind CSS + Lovable Cloud backend.

## üéØ COMMON MODIFICATIONS

### To Change Background Color:
1. Open `src/index.css`
2. Find line ~12: `--background: 45 50% 98%;`
3. Modify the HSL value

### To Change Primary (Forest Green) Color:
1. Open `src/index.css`
2. Find line ~20: `--primary: 109 38% 25%;`
3. Modify the HSL value

### To Change Accent (Harvest Gold) Color:
1. Open `src/index.css`
2. Find line ~35: `--accent: 42 87% 57%;`
3. Modify the HSL value

### To Add New Language:
1. Open `src/contexts/AppContext.tsx`
2. Add to `Language` type (line ~43)
3. Add translations object (starting around line ~52)
4. Update language options in `src/components/dashboard/Header.tsx`

### To Modify Crop Disease Database:
1. Open `src/components/dashboard/AIScanner.tsx`
2. Find `diseaseDatabase` object (line ~32)
3. Add or modify crop entries

### To Add New Shop Data:
1. Use database migration tool or Lovable Cloud UI
2. Insert into `shops` table

## üóÑÔ∏è DATABASE SCHEMA

**profiles** table:
- id (UUID)
- user_id (UUID, references auth.users)
- full_name (TEXT)
- preferred_language (TEXT, default 'en')
- field_mode_enabled (BOOLEAN, default false)

**scans** table:
- id (UUID)
- user_id (UUID, references auth.users)
- crop_type (TEXT)
- diagnosis (TEXT)
- cause (TEXT)
- organic_cure (TEXT)
- chemical_cure (TEXT)
- confidence (DECIMAL)
- image_url (TEXT)
- healthy_comparison_url (TEXT)

**shops** table:
- id (UUID)
- name (TEXT)
- address (TEXT)
- latitude (DECIMAL)
- longitude (DECIMAL)
- phone (TEXT)
- pesticide_stock_list (TEXT[])
- organic_products (TEXT[])
- rating (DECIMAL)
